# packet switching
회선교환: 이미 점유한 길로만 감.
- 장점: 방해를 안 받음. 끊어질때까지
패킷 교환: 정해진 길이 없음. 고속도로처럼 어디가 막히면 우회
- 그때그때 최적화된 길로만 감.

딜레이가 있어서 router에 패킷이 쌓임 -> 로컬 버퍼.
큐가 다 차면 버려짐 -> 로스

routing: 패킷의 전체 길을 알음.
forwarding은 다음 로드에만 관심 있음.
